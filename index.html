🔥 한국 커뮤니티 애그리게이터

4개 주요 커뮤니티 사이트의 실시간 인기 게시물을 한눈에!

📋 프로젝트 소개

이 프로젝트는 한국의 주요 커뮤니티 사이트들의 인기 게시물을 자동으로 수집하여 한 곳에서 볼 수 있는 웹 애플리케이션입니다.

🎯 지원 사이트

🚗 보배드림 조회수 + 추천수 + 댓글수

자동차 및 종합 커뮤니티의 인기 게시물을 실시간 수집

💰 뽐뿌 조회수 + 추천수 + 댓글수

할인 정보 및 쇼핑 커뮤니티의 HOT 게시물 모음

🎮 fmkorea 추천수 + 댓글수

축구 및 종합 엔터테인먼트 베스트 게시물

💬 디시인사이드 조회수 + 추천수 + 댓글수

실시간 베스트 갤러리의 화제 게시물

🚀 주요 기능

  • ✅ 실시간 크롤링: 1시간마다 자동 업데이트
  • ✅ 인기도 정렬: 조회수 + 추천수×2 + 댓글수×3 알고리즘
  • ✅ 사이트별 필터링: 원하는 사이트만 선택 가능
  • ✅ 검색 기능: 제목 기반 실시간 검색
  • ✅ 반응형 디자인: PC/모바일 최적화

💻 기술 스택

Backend: Flask, SQLAlchemy, SQLite
Frontend: HTML5, CSS3, JavaScript
크롤링: BeautifulSoup4, Requests
배포: Vercel, GitHub Actions

📂 GitHub 저장소 🌐 라이브 데모 (준비중)

🛠️ 로컬 실행 방법

git clone https://github.com/reomoon/community.git
cd community
pip install -r requirements.txt
python run.py

실행 후 http://localhost:5000에서 확인 가능합니다.
